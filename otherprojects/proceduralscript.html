Procedural Generation

A python script for Autodesk Maya which generates a user interface for building pipe pieces through procedural generation. Solo project. Completed during April 2019.

Requirements

  • Create a user interface for procedural generation of chosen topic (pipes) for use of quicker, easier modeling of background elements in a 3D scene
  • Focus on the capabilities of the generator rather than a appealing GUI
  • Allow for changing of lengths, widths, and multiple segment types

Features

  • User interface for selection of variables
  • Create pipes of types: basic straight, straight connector, 45deg elbow joint, 90deg elbow joint, equal t-junction, unequal t-junction, equal cross fitting, unequal cross fitting, y-junction, and basic turning lever
  • Create pipes with textures: stainless steel, copper, white PVC, and black PVC
