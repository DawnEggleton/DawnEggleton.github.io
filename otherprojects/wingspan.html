Wingspan

A prototype controller using alternative sensor types and Arduino. Primary role as hardware creation and circuitry. Completed between October and December 2018.

Requirements

  • Use at least two sensors for control
  • Be able to use the controller to navigate a provided 3D world

Features

  • Stretch sensor to control the speed
  • Tilt sensors to control left, right, up, and down motion
  • Finished and smoothed edges for comfort, despite prototype status
