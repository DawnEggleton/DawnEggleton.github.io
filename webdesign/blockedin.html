Blocked In

A Tumblr theme with additional page designs for a character grid and character page. Designed for storing content across multiple writing sites for multiple characters while still keeping characters which match certain specifications easy to find and filter out from the others. Uses Isotope.js. Intended for personal use. Can be seen live here.

Features

  • Isotope.js filtering on the character grid
  • Tabbed content on the character pages for instances of multiple site usage of a single character
  • Feature image changes based on page URL for templated pages where this is not in my control due to Tumblr's code structure (e.g., all pages with /tagged in the url)
